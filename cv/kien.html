Ứng Dụng Quản Lý Quy Trình Làm Việc

06/2025 - Hiện tại

Dự án cá nhân đang phát triển, xây dựng một công cụ quản lý công việc theo phong cách Kanban Board (Trello/Jira) để giúp các nhóm nhỏ tối ưu hóa quy trình làm việc.

  • Thiết kế và triển khai giao diện người dùng với chức năng kéo-thả (drag-and-drop).
  • Xây dựng API cho các chức năng quản lý bảng, danh sách và thẻ.
  • Sử dụng AJAX để cập nhật trạng thái công việc theo thời gian thực.
Công nghệ: PHP, MySQL, JavaScript (AJAX), HTML5, CSS3.

Ứng Dụng Web Quản Lý Chăm Sóc Cá Koi

01/2025 - 04/2025

Hệ thống web giúp người dùng theo dõi và quản lý các thông số quan trọng khi nuôi cá Koi tại nhà thông qua trình duyệt, có thể truy cập từ mọi thiết bị.

  • Phát triển các chức năng backend bằng C# với kiến trúc MVC trên nền tảng ASP.NET Core.
  • Xây dựng giao diện người dùng (UI) responsive bằng Razor Pages để hiển thị biểu đồ và dữ liệu.
  • Sử dụng Entity Framework Core để tương tác với cơ sở dữ liệu SQL Server, quản lý dữ liệu về hồ cá và lịch chăm sóc.
Công nghệ: C#, ASP.NET Core, Entity Framework Core, SQL Server.
